# 🚀 My Developer Portfolio

Welcome to my personal developer portfolio web application! This project showcases my journey, skills, credentials, and projects as an aspiring web developer at Life Choices Academy.

## 🔗 Live Demo
Check out the live website here: https://nqenqaanela-beep.github.io/portfolio/

---

## 🎨 Project Overview
This portfolio is built to serve as a digital CV and a hub for all my frontend and programming accomplishments. It features a fully responsive design, a dynamic contact form, and smooth navigation sections highlighting my technical skills and badges.

### Core Sections:
- **Home / Hero:** A welcoming introduction to who I am.
- **About Me:** Insights into my coding passion and background.
- **Tech Stack:** A clean grid layout highlighting the tools I use.
- **Projects & Badges:** Features my academy milestones, including Cisco Networking Academy modules.
- **Get In Touch:** A functional contact form connected to a backend notification system.

---

## 🛠️ Built With

This project was built entirely using core frontend web technologies:

* **HTML5:** For creating semantic, structured, and accessible page elements.
* **CSS3:** For custom styling, layouts (Flexbox/Grid), animations, and dark-theme aesthetics.
* **Web3Forms API:** Integrated to handle seamless contact form email submissions.

---

## 📱 Features

- **Fully Responsive Layout:** Optimized to look great on desktop monitors, tablets, and mobile smartphones.
- **Interactive Forms:** Input field validation with real-time status updates and submission handling.
- **Smooth Anchor Navigation:** Clean navbar scrolling that smoothly guides users directly to designated sections.
